Configuracion de Vim mas Neovim

1 Paso para habilitar funciones de vim que se encuentra desactivados de forma  predetermidanos.

set number 
set mouse=a
set numberwidth=1
set clipboard=unnamed
syntax on
set showcmd
set ruler
set cursorline
set encoding=UTF-8
set showmatch
set sw=2
set relativenumber

2. Enlacar las carpetas de plugin y mapeo de teclado

so ~/.vim/plugins.vim
so ~/.vim/plugin-conf.vim
so ~/.vim/map.vim

3. Habilitar las configuraciones de los plugins

colorsheme gruvbox
let g:gruvbox_contrast_dark = "hard"
highlight Normal ctermbg=NONE
set laststatus=2
set noshowmode
